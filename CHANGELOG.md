# Changelog

Все значимые изменения в проекте документируются в этом файле.

## [2026-04-17] - Исправления и улучшения

### Исправлено
- **Event loop is closed** - Исправлена ошибка в webhook handler, теперь правильно переиспользуется event loop
- **Дублирование записей в Google Sheets** - Добавлен трекер последних сохранений (10 секунд) для предотвращения дубликатов на уровне пользователя
- **Избыточное сообщение о дубликате** - Убрана проверка дубликатов из `save_to_sheet`, оставлена только проверка через трекер
- **Парсинг SMS** - Попытка переключения на OpenRouter не удалась из-за ограничений PythonAnywhere, вернулись к Gemini

### Добавлено
- **Трекер сохранений** - Глобальный словарь `last_save_tracker` для отслеживания последних операций сохранения по пользователям
- **Улучшенные сообщения об ошибках квоты** - Профессиональные сообщения с информацией о времени сброса квоты (полночь Pacific Time)
- **Подробное логирование** - Добавлены логи для диагностики:
  - `[CALLBACK]` - события callback handlers
  - `[SAVE]` - операции сохранения в Google Sheets
  - `[TRACKER]` - отслеживание операций сохранения
  - `[SMS PARSE]` - парсинг SMS
  - `[GEMINI SMS]` - ответы от Gemini для SMS
  - `[EDIT]` - редактирование данных пользователем
  - `[DUPLICATE BLOCKED]` - блокировка дубликатов
- **Сброс флага processing** - После редактирования флаг `processing` сбрасывается в `False`

### Изменено
- **Приветственное сообщение** - Упрощено с 13 строк до 5, более лаконичное и профессиональное
- **Timezone** - Все записи теперь сохраняются с временем UTC+5 (Узбекистан)
- **Версия бота** - `BOT_VERSION = "2.0.0-inline-buttons"`

### Удалено
- **Лишние файлы документации** - Удалены `DEPLOY_INSTRUCTIONS.md`, `FINAL_FIX_INSTRUCTIONS.md`, `QUICKSTART.md`
- **Co-Authored-By строки** - Переписана история git для удаления Claude из Contributors
- **OPENROUTER_API_KEY** - Убрана неиспользуемая переменная из кода

### Технические детали

#### Защита от дубликатов
Двухуровневая система:
1. **Трекер на уровне пользователя** - блокирует сохранение одинаковых данных в течение 10 секунд
2. **Флаг processing** - предотвращает двойные клики на кнопку подтверждения

#### Модели AI
- **Фото чеков**: `gemini-2.5-flash` через Google AI Studio
- **SMS**: `gemini-2.5-flash` через Google AI Studio (та же модель)
- **Попытка OpenRouter**: Не удалась из-за ограничений сети PythonAnywhere (бесплатный аккаунт)

#### Квоты Gemini API
- Бесплатная квота сбрасывается ежедневно в полночь по тихоокеанскому времени (Pacific Time)
- Лимиты можно проверить на https://aistudio.google.com/rate-limit

### Известные ограничения
- PythonAnywhere (бесплатный аккаунт) блокирует исходящие соединения к сайтам вне белого списка
- OpenRouter недоступен на бесплатном аккаунте PythonAnywhere
- Все запросы (чеки + SMS) используют одну квоту Gemini API
