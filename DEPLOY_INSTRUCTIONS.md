## Шаги для развертывания на PythonAnywhere:

### 1. Загрузите файлы
```bash
# Загрузите на PA:
- bot.py
- mapping_service.py
- .env
- credentials.json
- requirements.txt
```

### 2. Установите зависимости
```bash
pip install --user -r requirements.txt
```

### 3. Настройте WSGI файл
Создайте файл `/var/www/la488_pythonanywhere_com_wsgi.py`:

```python
import sys
import os

# Добавьте путь к вашему проекту
project_home = '/home/la488/Tg-bot-Check-Converter'
if project_home not in sys.path:
    sys.path.insert(0, project_home)

# Установите переменную окружения для определения PA
os.environ['PYTHONANYWHERE_DOMAIN'] = 'la488.pythonanywhere.com'

# Импортируйте Flask app
from bot import app as application
```

### 4. Настройте переменные окружения
В консоли PA выполните:
```bash
export PYTHONANYWHERE_DOMAIN=la488.pythonanywhere.com
```

### 5. Перезагрузите веб-приложение
В разделе "Web" нажмите "Reload"

### 6. Проверьте логи
Если бот не работает, проверьте:
- Error log: `/var/log/la488.pythonanywhere.com.error.log`
- Server log: `/var/log/la488.pythonanywhere.com.server.log`

## Проверка работы:

1. Отправьте боту фото чека
2. Бот должен распознать данные и показать кнопки подтверждения
3. Нажмите "✅ Все верно" - данные сохранятся в таблицу
4. Отправьте SMS текст (длиннее 20 символов)
5. Бот распознает и попросит подтверждение

## Возможные проблемы:

### Бот не отвечает:
- Проверьте webhook: `https://la488.pythonanywhere.com/mysecrettoken123`
- Убедитесь что WSGI файл настроен правильно
- Проверьте логи ошибок

### Ошибка "Event loop closed":
- Уже исправлено в коде (используется новый bot для каждого запроса)

### Ошибка доступа к Google Sheets:
- Убедитесь что credentials.json загружен
- Проверьте что сервисный аккаунт имеет доступ к таблице
- Email сервисного аккаунта должен быть добавлен в "Настройки доступа" таблицы

### Quota exceeded (лимит Gemini):
- Бесплатный лимит: 15 запросов в минуту
- Подождите 24 часа или обновите API ключ
