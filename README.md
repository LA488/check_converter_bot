# Telegram Bot - Сканер чеков

Бот для распознавания чеков и SMS с автоматической категоризацией расходов.

## Возможности:
- 📸 Распознавание фото чеков через Gemini AI
- 💬 Парсинг SMS от банков
- ✅ Подтверждение данных перед сохранением
- 📊 Автоматическое сохранение в Google Sheets
- 🔍 Поиск по брендам, юр.лицам, категориям

## Команды:
- /start - Главное меню
- /reload - Обновить базу брендов
- /help - Помощь

## Кнопки подтверждения:
- ✅ Все верно - сохранить данные
- ✏️ Редактировать - изменить данные
- ❌ Отмена - отменить операцию
